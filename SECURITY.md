# Security Policy

## Supported Versions

Only the latest release is supported: **v0.1.0**.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.0   | :white_check_mark: |


## Reporting a Vulnerability
Please report security issues privately via **GitHub Security Advisories**:
Repo -> **Security** -> **Report a vulnerability**.

If you can’t use that, open an issue **without sensitive details**.

Include: version, OS, install method (cargo/AUR/Homebrew/binary), and steps to reproduce.
